# Smart Number Analyzer

C++ console application that analyzes integer properties and provides statistical insights.

## Features

-Checks if a number is palindrome
-Detects Armstrong numbers
-Calculates digit sums
-Finds numbers with more even than odd digits
-Determines the number with the maximum digit sum
-Computes the average of all numbers

## How It Works

The program reads a list of integers and performs multiple analyses on them.

## Example

Input:
5
121 45 153 88 10

Output:
Palindromes: 2
Armstrong numbers: 1
Number with max digit sum: 88
Numbers with more even digits: 1
Average: 83.40
